Thesis

During my undergraduate studies, I completed my thesis titled “A Notification Based Non-Restrictive Intervention Approach for Handling Excessive Smartphone Use” under the supervision of Marium-E- Jannat. The research explored how awareness-based notifications can help mitigate excessive smartphone usage without imposing restrictive controls. As part of the study, I conducted an online survey to examine users’ awareness of their smartphone usage patterns and developed an Android application called App Usage Tracker to monitor usage behavior. A three-week experiment was then conducted using this application to collect real-world smartphone usage data and evaluate the effectiveness of the proposed non-restrictive intervention approach.

Projects

App Usage Tracker [GitHub]

An Android application that tracks the usage of all installed apps on a smartphone. Users can view daily and weekly usage history and set usage targets to help mitigate excessive smartphone use.

IAM System with RBAC using Spring Security [GitHub]

A secure Identity and Access Management (IAM) system implemented using Spring Boot and Spring Security. The system supports authentication, authorization, RBAC, and MFA. Security hardening techniques were applied to mitigate threats such as brute-force attacks and privilege escalation.

Technologies: Java, Spring Boot, Spring Security, JDBC, Hibernate (JPA)

Fredericton Cycling Infrastructure Analysis [StoryMap]

An interactive ArcGIS StoryMap analyzing population density, cycling infrastructure, transit stops, parks, and road networks in Fredericton. The project identifies priority areas for improving safe cycling infrastructure using spatial analysis and GIS visualization techniques.

Tools: ArcGIS Pro, ArcGIS StoryMaps, Spatial Analysis
